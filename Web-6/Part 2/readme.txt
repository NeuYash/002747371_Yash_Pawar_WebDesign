Part B: Async/Await/Promises/Set Interval/Clear Interval to be used to Design a Simple Stop Watch Web Application.


1) It is a Single Page Application. It should have one Text Field/Label and Three Buttons.

2) Text Field/Label Should not be editable. It should display the Time in "HH:MM:SS" format. Initially, it should display 00:00:00

3) Three Buttons are “Start, Stop and Reset”. On Click of Start, Time should start running. On Click of Stop, time should be Paused. On Click of Reset. Time to be reset to 00:00:00.
